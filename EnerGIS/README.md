# EnerGIS

**Role:** System Analyst  
**Focus:** System architecture, ERD, Use Case, Activity Diagram, UI/UX  

  ## Description
An educational application that also maps renewable energy in Indonesia, particularly power plants that use renewable energy sources. EnerGIS aims to increase the knowledge and awareness of the Indonesian people regarding renewable energy and its application in Indonesia, including the types of renewable energy, energy conversion, and its role in everyday life. My Tasks:
- Designed and documented system interfaces (Homepage, Map Page, Chatbot, Admin Dashboard) based on detailed functional requirements and user flows.
- Performed UI validation and functional testing to ensure alignment with specifications, usability standards, and overall system performance.
- Collaborated in requirements analysis and process mapping to support accurate system design and implementation.
